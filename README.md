# Прототип магазина кроссовок на React

Прототип интернет-магазина кроссовок на React + TS + Redux toolkit + firebase, изначально проект был написан на react + js + react context + mockapi, спустя год я к нему вернулся и переписал на более актуальный стэк. 

Данный проект - это мой тестовый полигон, в планах написать самому бэкенд для данного проекта.

Стэк: 
+ React;
+ Typescript;
+ Redux toolkit;
+ Google firebase;
+ LocalStorage.
